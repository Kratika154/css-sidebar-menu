# Responsive Sidebar Menu

A modern and responsive sidebar navigation menu built using HTML and CSS.

## 🚀 Features

* Responsive sidebar navigation
* Smooth open and close animation
* Font Awesome icons
* Social media integration
* Clean and modern UI
* Pure HTML and CSS (No JavaScript)

## 🛠️ Technologies Used

* HTML5
* CSS3
* Font Awesome
* Google Fonts (Poppins)

## 📂 Project Structure

